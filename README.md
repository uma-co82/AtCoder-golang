### Set Up
＊ 「go version go1.13.4」 confirms operation.

1. go get this repo

```shell
go get -u github.com/uma-co82/AtCoder-golang
cd $GOPATH/src/github.com/uma-co82/AtCoder-golang
go run test.go
```
